next主题的安装

title: next主题的安装
date: 2016-11-22 20:10:01

tags:

已经是安装了hexo+guihub博客的第三天了。前天是因为太晚了，匆匆的安装完就没继续了。昨天也只是简单的写了点所思所想的矫情。今天终于开始好好的改造我的博客了。
#
别人推荐的用next主题。然后各种百度各种找资料。

1：先在git bash中输入 git clone https://github.com/iissnan/hexo-theme-next themes/next。就安装好了next。

2：然后在站点配置文件下，找到theme，改变其值为next。

3：最好在用hexo clean清除缓存，然后hexo s –debug调试。会发现出现了http://localhost:4000这个站点信息。从这里开始，我就开始蒙逼 了。因为我直接复制了这个网站，在浏览器上粘贴，发现出现无法显示网页。我重复弄了好多次。然后我百度了，有的说，是因为后面这个4000被占用了，但是我没有理解是什么意思等等，有的是我不理解的答案，有的解释根本没有用。然后我也不知道怎么弄的，我重新弄一次之后，复制粘贴，又有用了。

然后我又发现了一个问题。我输入http://localhost:4000，只能当我重新用git bash输入hexo s，命令中的http://localhost:4000我复制粘贴才有用，要是直接默认的话还是会显示网页不存在。但是我要是输入https://chenjiatai.github.io，并不会出现我改的next主题。这里我也还没搞懂。然后通过百度，我发现http://localhost:4000是只能自己看到，并没有传到github上去，因此要输入hexo g和hexo d。这个是提交到github的命令。

然后在next主题的_config.yml中，通过修改配置来变化。到此，next主题弄好了

唉，别看我现在说的这么简单，鬼知道我查找了多少资料。感觉每次使用博客我都能遇到各种困难，各种问题。我差点就崩溃了。算了，不足道也。完成就行，仅此记录我的第一篇实用的博客。欧耶
